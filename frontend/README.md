# 🎨 Frontend – CredTrade AI Agent

This folder contains the frontend interface for the CredTrade AI Agent.

---

## 🧠 Purpose

The frontend visualizes:

- Trader leaderboard  
- Credibility scores  
- Trader classification (Skilled / Opportunistic / High Risk)

---

## ⚙️ Features

- 📊 Leaderboard UI  
- 🧾 Trader profile view  
- 📈 Score visualization  

---

## 🔌 Integration

The frontend is designed to consume data from:

- CredTrade AI scoring engine (backend)
- Pacifica APIs (future integration)

---

## 🚀 Future Improvements

- Real-time leaderboard updates  
- Wallet-based trader tracking  
- Interactive analytics dashboard
